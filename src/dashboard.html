1.3.4

  • Fixed downloads from F and G servers.
  • Removed downloads from RapidVideo due to a recent site change. (Will be re-added once a fix is found).

1.3.3

  • Reverted most of the changes from the previous release because it was not a very reliable way to deal with the anti adblock (I still haven't found a way). For the time being if you see a please disable adblock message, goto 9anime Companion Settings > disable Remove Ads and Remove Social Share.
  • Removed the Github permission.

1.3.2 (emergency release)

  • Fixed the please disable adblock message that appear on the player when Remove Ads is enabled.
  • Added a way to remotely patch please disable adblock message. If you see it again, remember to notify me on github, wait for me to patch and then go to Settings > Anti Adblock Detection > Update. But remember, please dont spam the update button. Try once every 10/15 mins if you have to.

1.3.1 (emergency release)

  • Fixed the please disable adblock message that appear on the player when Remove Ads is enabled.

1.3.0 (hotfix release)

  • Added 3rd party iframe removal.
  • Added fix for the new popups and ads.

1.2.0

  • Added 9anime.ru to the list of supported domains.
  • Added bebi.com to the ad filter list.
  • Changed the way MAL autoupdate works. Now it will only update if the current episode is higher than your MAL episode.

1.1.0

  • Added support for downloading from G servers. (G1, etc)
  • Initial implementation of Watch on 9anime button within MyAnimeList.net (Screenshot)
  • Changed the search url format for Reddit Discussion. The results should be much better now.

1.0.0

  • Changed the color and effect of the switches in the dashboard. Thanks @WEBhicham.
  • Added Keyboard Shortcuts to settings. Find it under General settings.
  • Added support for downloading from RapidVideo.
  • All the modals now have a "X" button.
  • Changed scroll to player to scroll to top. Added a smooth scroll animation to it and made it global. The button and shortcut can now both be used throughout 9anime.
  • Changed native browser notifications to in site notifications.

1.0.0-beta.2

  • Added auto-update for the MyAnimeList integration. This can be found in the MyAnimeList configure box.
  • Fixed an issue which was causing the alternate search overlay to pop up when typing password and in the report episode box.

1.0.0-beta.1
  • Redesigned the popup. Thanks @densityx.
  • Improved the downloader. Now 9anime Companion can download from all the 9anime Servers (F2, F4 etc).
  • Redesigned the download modal. Thanks @yumiris.
  • Vastly improved the ad/popup blocker. The new ad remover uses a combination of web request blocking and page element removal/function overrides. In my tests it removed 100% ads/popups without the need for an external adblocker.
  • Added support for keyboard shortcuts.
  • Added alternate search overlay . Shortcut s.
  • Added scroll to player. Shortcut t.
  • Added recently watched anime. This can be accessed via the popup.
  • Vastly improved the core functionality of MyAnimeList integration. It can easily be expanded in the future to add features like Auto MAL sync, full featured add/edit etc
